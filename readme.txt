== Description ==

Creates a sidebar widget that allows site users to add themselves to a blog based on predefined conditions. Based heavily on the sidebar add users widget by DSader

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the add-users-sidebar-widget.zip file to a directory of your choice(preferably the wp-content/plugins/widgets folder)

2. Unzip the add-users-sidebar-widget.zip file into the wordpress plugins/ directory: 'wp-content/plugins/'

3. Activate the 'Add Users Sidebar Widget' plugin from the Plugins' menu in WordPress

4. Access the widgets menu and add the add-link widget to your pages.
